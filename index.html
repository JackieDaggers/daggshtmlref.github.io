HTML Notes

Tables

Tag Name Use
<table> Table Creates HTML table
<thead> Table Head Contains the row of column headers
<tbody> Table Body Contains all of the other rows of the table
<th> Table Heading Cell for Headers and first cell in row
<tr> Table Row Generations a row in the table
<td> Table Data Creates individual cells of the table. # of TDs determine row length/
<tfoot> Table Foot Contains footer. Add <tr> to create row
coldspan Column Span Defines how many columns a cell takes up. ="#
rowspan Row Span Defintes how many rows a cell takes up. ="#

Handle all borders via CSS, rather than depreciated inline

Semantic HTML

Tag Name Use
<header> Header Note: Can exist in article
<article> Article Independent section, works outside of page context. Page can have multiple (eg blog posts)
<section> Section Sections off page into related bits, not for layout (use <div>
<nav> Navigation Note: Can have mult per page, fxn beyond header
<footer> Footer Note: Can exist within article
<aside> Aside Contextual break, good for things like quotes, stats, sidebars
<time> Time Tags parent with metadata, not a head element, but used in SEO
<address> Address Tags parent with metadata, author contact info
<figure> Figure Insert figure image into article/body.
<figcaption> Figure Caption Caption displayed (typ. below) figure. Brief description/explanation
datetime Timestamp Attribute datetime='ymd xx:xx(24hr)+/-0000(timezone)' time and tz optional
alt Text Alternate Text replacement for figure img, not always necessary (can just use <figcaption>

Pseudo Semantic

Audio: src type='audio/type controls etc.
Video: src controls Video Not Supported
Lang: ='xx' use in span to indicate text is pronounced differently than rest of page

CSS Notes

Custom Fonts

Link via html head, use @font-face in css

  • html links pulled from online font source (eg google fonts)
  • font-face links in css using local path or latin source info grabbed from font url

Deepest Browser Support

@font-face {
 font-family: 'MyWebFont';
 src: url('webfont.eot'); /* IE9 Compat Modes */
 src: url('webfont.eot?#iefix') format('embedded-opentype'), /* IE6-IE8 */
   url('webfont.woff2') format('woff2'), /* Super Modern Browsers */
   url('webfont.woff') format('woff'), /* Pretty Modern Browsers */
   url('webfont.ttf') format('truetype'), /* Safari, Android, iOS */
   url('webfont.svg#svgFontName') format('svg'); /* Legacy iOS */
}

Slightly Deep Broswer Support

@font-face {
 font-family: 'MyWebFont';
 src: url('webfont.woff2') format('woff2'), /* Super Modern Browsers */
   url('webfont.woff') format('woff'), /* Pretty Modern Browsers */
   url('webfont.ttf') format('truetype'), /* Safari, Android, iOS */
}

Positioning and Display

Property Value Use
position
static in place, item won't be moved by css rules
relative positions relative to default position, moveable
absolute remove from flow, placed relative to closest parent element
fixed remove from flow, locked in place (can't scroll past)
sticky relative in document until specified position is reached then sticks in place
display
inline in line with content around them, can't adjust h/w, use for things like h, p, div to make inline
block given own line apart from surrounding content, use for em, strong
inline-block brings elements inline while allowing for height/width manipulation
z-index # element depth on page, places overlapping elements above/blow others, doesn't work on static elements
float left/right moves element as far left/right as possible
clear l/r/both/none changes behavior when inline-block elements bump into each other. Value sets where item can't be touched, operates within containing element
t/b/l/r #px offset #px from selected side
box-sizing
content-box default box model
border-box w/h set box maximums, content auto resized to accommodate borders, padding
